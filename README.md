💼 Noman Awan - Portfolio Website

This is the official portfolio website of Noman Awan, showcasing skills, projects, contact information, and professional background. 
The website is built using modern web technologies with a focus on responsive design, personal branding, and accessibility.

🌐 Live Website
🔗 https://portfolio-website-new-five.vercel.app/

✨ Features

• Responsive design for mobile, tablet, and desktop
• Clean and modern UI/UX
• Dedicated sections: Home, About, Skills, Projects, Contact
• Interactive project showcase with live previews
• Social media integration
• Downloadable resume/CV option
• Smooth scrolling and animations

🧰 Technologies Used

• HTML5 – for content structure
• CSS3 – for design and layout
• JavaScript – for interactivity and DOM manipulation
• GitHub Pages – for hosting and deployment
• Google Fonts – for typography
• Font Awesome – for icons
