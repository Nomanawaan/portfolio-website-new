Project Title: Etsy – Handmade Marketplace Clone

Tech Used: HTML, CSS, JavaScript

Live Preview: View Website

GitHub: Source Code

Description

This project is a front-end clone of the popular Etsy marketplace, focusing on showcasing handmade, vintage, and unique goods.
